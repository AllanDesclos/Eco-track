# Eco-track

Description :

EcoTrack est une application web dédiée au calcul et au suivi des émissions de CO₂ liées aux déplacements.
Le site permet aux utilisateurs d’estimer leur empreinte carbone annuelle en fonction de plusieurs paramètres : type de carburant, distance parcourue, nombre de trajets et année de référence.
